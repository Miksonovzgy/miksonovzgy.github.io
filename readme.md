## What is this?

This applications goal is to teach concepts of Computer Science in a visual and interactive way. It currently includes Kruskals and Prims algorithm.

[DEMO](https://ct-algos.web.app/)

## How to deploy?

```
npm install
npm run start //for local development
npm run build //to make a deployable build
```

The application is currently deployed on Google Firebase and I left the configuration files for Google Firebase for [easy deployment](https://firebase.google.com/docs/hosting/quickstart?hl=en&authuser=0).

[NodeJS](https://nodejs.org/en/) and npm needs to be installed on the system.

## How to contribute?
